Convert Text to Numeric

Translate text-based values ("Seventy five", "~120", "about eight") into clean numeric fields using AI so they can be symbolized or analyzed.

Usage

  1. Add the toolbox to ArcGIS Pro.
  2. Open Convert Text to Numeric.
  3. Pick the provider/model, set the layer and field, then run the tool.

Parameters

NameTypeRequiredDescription
SourceString (Choice)YesProvider such as OpenRouter, OpenAI, Azure OpenAI, Claude, DeepSeek, Local LLM.
ModelStringNoSpecific model for the provider.
EndpointStringNoCustom endpoint (Azure/local).
Deployment NameStringNoAzure OpenAI deployment ID.
Input LayerFeature LayerYesLayer containing the text field.
FieldFieldYesField to convert to numeric.

Tips

  • Spot-check a subset of records before trusting the full conversion.
  • Use the same model you used to generate text so tone/format is familiar.
